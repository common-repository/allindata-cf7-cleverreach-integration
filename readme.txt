Push Contact Form 7 information to CleverReach

== Description ==

Contact Form 7 is a widely used Wordpress Plugin to manage multiple contact forms. CleverReach is a Software-as-a-Service
provider for newsletter systems.

This plugin allows to map contact form data of Contact Form 7 to receiver list information and push it automatically to
CleverReach. Now you can easily build up landingpages and newsletter subscriptions without the need to use different
plugins to create contact forms.

== Features ==

* Enable or disable the CleverReach Integration for each contact form individually
* Configure the data mapping for each form
* Create your own CleverReach API v3 OAuth application credentials and connect your wordpress instance to it
* CleverReach API Key can be fetched with just a click

== Requirements ==

You have to use the Wordpress plugin [Contact Form 7](https://de.wordpress.org/plugins/contact-form-7/) and an active
account for [CleverReach](https://www.cleverreach.com) is required.

== Installation ==

* Upload the entire `allindata-cf7-cleverreach-integration` folder to the `/wp-content/plugins/` directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* General settings can be configured in 'Settings > All.In Data - Contact Form 7 CleverReach Integration'
* Form settings can be configured in each Contact Form 7 by navigating to the new tab 'CleverReach Integration'
